# Gym Mobile App (MobDevApp)

A comprehensive workout and fitness tracking mobile application built with React Native and Expo.

## Features

- **Workout Tracking**: Log and monitor your daily exercises.
- **Calendar Integration**: View your workout history and schedule with an interactive calendar.
- **User Profiles**: Manage your fitness profile and progress.
- **Modern UI**: Clean, intuitive, and responsive design for a premium user experience.

## Tech Stack

- **Framework**: [Expo](https://expo.dev/) / [React Native](https://reactnative.dev/)
- **Navigation**: [React Navigation](https://reactnavigation.org/)
- **Icons**: [Expo Vector Icons](https://docs.expo.dev/guides/icons/)
- **Components**: React Native Calendars, SVG support, etc.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (LTS version recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- [Expo Go](https://expo.dev/expo-go) app on your mobile device (for testing)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AquaDarknessMegumin/Gym-Mobile-App.git
   cd Gym-Mobile-App
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

### Running the App

- **Start the development server:**
  ```bash
  npm start
  ```
- **Run on Android:**
  ```bash
  npm run android
  ```
- **Run on iOS:**
  ```bash
  npm run ios
  ```
- **Run on Web:**
  ```bash
  npm run web
  ```

Once the development server is running, you can scan the QR code displayed in your terminal using the Expo Go app to view the app on your physical device.
